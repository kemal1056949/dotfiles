dotfiles
========

My config files for bash, vim, etc. Currently this repository only has `.bash_aliases`, `.bash_ps1`, `.bashrc`, and `.vimrc`.

Installation
------------

1. Clone this repository.
2. Execute `install.sh` script to install dotfiles.
